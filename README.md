# HELM - Navigation made simple

Professional boat navigation app for everyone.

## Features

- Offline charts and navigation
- 6000+ harbors & anchorages database
- Smart route planning
- Community reviews & photos
- Real-time weather integration
- Works on all devices

## Tech Stack

**Frontend:**
- HTML5 / CSS3 / JavaScript ES6+
- Progressive Web App (PWA)
- Responsive Design

**Backend (Q2 2026):**
- Node.js + Express
- PostgreSQL + PostGIS
- RESTful API

**Data Sources:**
- OpenSeaMap (Free nautical charts)
- Global Fishing Watch (Anchorages)
- OpenStreetMap (Marine features)

## Project Timeline

- **Q1 2026:** Base App + Landing Page
- **Q2 2026:** Backend + User System
- **Q3 2026:** Mobile App + Offline Features
- **Q4 2026:** AIS Integration
- **2027+:** Commercial Features
